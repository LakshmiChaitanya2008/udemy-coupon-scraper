# Udemy Coupon Scraper

A node.js script that scrapes various websites to gather free coupon codes for Udemy courses

Scrapes coupons from:

1. Udemy Freebies
2. Tutorial Bar
3. Real Discount
4. IDownloadCoupon
5. CourseVania
6. E-Next
